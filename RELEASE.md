# Release process

This is here because I forget things sometimes. :-P

## Steps

1. Add the name and version of the system to the [README](README.md).
1. Update the documentation if necessary.
1. Merge `develop` to `master`.
1. Run the [Release Action](.github/workflows/release.yaml).
1. Update the [project page](https://foundryvtt.com/packages/loot-parcels/edit).
